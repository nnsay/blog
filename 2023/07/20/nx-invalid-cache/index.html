3. 总结

Nx 计算 Hash 直接影响缓存的使用, 在最新的版本中有加入的机器 ID 的检查, 所以本地缓存复用会变的更困难, Nx 的 Hash 计算和缓存复用至少在同命令同机器上好用的, 但是需要注意缓存失效及其容易发生, 缓存管理不透明也不好控制, 希望通过缓存加速的可以有限使用, 比如同 PR 同流水线同 Base 缓存命中是好用的.
