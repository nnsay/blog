1. PicGo-Core

A tool for pictures uploading. Both CLI & API supports

PicGo-Core其实PicGo下的一个项目, PicGo是一个基于Electron的GUI客户端而PicGo-Core是做图床实现的核心,如其GitHub的描述, Core只是CLI和API, 其中API就是NPM package, API方式使用时更像是一个做图床的SDK. 本问主要介绍CLI的使用.
